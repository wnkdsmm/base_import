Требования:

1. Операционная система Windows 10/11
2. Python версии 3.10 или выше
3. Подключение к интернету (необходимо только при первой установке зависимостей)
4. PostgreSQL (в случае использования базы данных в качестве источника данных)

---

АВТОМАТИЧЕСКИЙ ЗАПУСК (двойной клик)

1. Откройте директорию проекта.
2. Выполните двойной щелчок по файлу `start_web_app.vbs`.
3. При первом запуске автоматически выполняются следующие действия:

   * создание виртуального окружения `.venv`;
   * установка зависимостей из файла `requirements.txt`;
   * создание файла `.env` на основе `.env.example` (при его отсутствии);
4. Дождитесь автоматического открытия приложения в браузере.

Если браузер не открылся:

* проверьте файл журнала: `logs\startup.log`

---

РУЧНАЯ УСТАНОВКА

1. Запустите файл `setup.bat`.
2. После завершения установки выполните запуск через `start_web_app.vbs`.

---

НАСТРОЙКА БАЗЫ ДАННЫХ

Параметры указываются в файле `.env`:

* `DATABASE_URL=postgresql://пользователь:пароль@хост:5432/имя_базы`
* `APP_HOST=127.0.0.1`
* `APP_PORT=8000`

Пример конфигурации:
`DATABASE_URL=postgresql://postgres:1234@localhost:5432/fires`

---

СТРУКТУРА ХРАНЕНИЯ ДАННЫХ

* Входные файлы размещаются в каталоге: `data\uploads\`
* Результаты обработки сохраняются в каталоге: `data\results\`

Каталоги `data\uploads` и `data\results` не следует удалять.

---

ПЕРЕНОС НА ДРУГОЕ УСТРОЙСТВО

1. Скопируйте каталог проекта целиком.
2. Установите Python версии 3.10 или выше на новом устройстве.
3. Откройте проект и выполните запуск через `start_web_app.vbs`.
