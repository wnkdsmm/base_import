Fire Data - очень простой запуск

Что нужно:
1) Windows 10/11
2) Python 3.10 или новее
3) Интернет (только для первой установки пакетов)
4) PostgreSQL (если у вас данные берутся из БД)

--------------------------------------------------
САМЫЙ ПРОСТОЙ ВАРИАНТ (двойной клик)
--------------------------------------------------
1) Откройте папку проекта.
2) Дважды кликните start_web_app.vbs.
3) При первом запуске он сам:
   - создаст .venv
   - установит зависимости из requirements.txt
   - создаст .env из .env.example (если .env нет)
4) Дождитесь открытия браузера.

Если браузер не открылся:
- смотрите лог: logs\startup.log

--------------------------------------------------
РУЧНАЯ УСТАНОВКА (если хотите сами)
--------------------------------------------------
1) Запустите setup.bat
2) После успешной установки запустите start_web_app.vbs

--------------------------------------------------
НАСТРОЙКА БД
--------------------------------------------------
Файл .env:
- DATABASE_URL=postgresql://пользователь:пароль@хост:5432/имя_базы
- APP_HOST=127.0.0.1
- APP_PORT=8000

Пример:
DATABASE_URL=postgresql://postgres:1234@localhost:5432/fires

--------------------------------------------------
ЧТО КУДА КЛАДЁТСЯ
--------------------------------------------------
- Входные файлы: data\uploads\
- Результаты: data\results\

Папки data\uploads и data\results не удаляйте.

--------------------------------------------------
ПЕРЕНОС НА ДРУГОЕ УСТРОЙСТВО
--------------------------------------------------
1) Скопируйте весь проект.
2) На новом ПК установите Python 3.10+.
3) Откройте проект и запустите start_web_app.vbs.

Готово.
