# Fire Analytics Dashboard — Инструкция по запуску

## Что нужно установить (один раз)
- [ ] Python 3.9 или новее — скачать: https://python.org/downloads/
      (при установке поставить галочку "Add Python to PATH")
- [ ] PostgreSQL — скачать: https://www.postgresql.org/download/windows/

## Первый запуск (один раз)
1. Убедитесь что PostgreSQL запущен и база данных создана
2. Дважды кликните на файл **setup.bat**
3. Следуйте инструкциям на экране
4. Когда откроется блокнот с файлом .env — проверьте настройки подключения

## Обычный запуск
Дважды кликните на файл **start_web_app.cmd**
Браузер откроется автоматически.

## Настройки подключения к базе данных (файл .env)
DATABASE_URL=postgresql://[пользователь]:[пароль]@[адрес]:[порт]/[база]

Пример: postgresql://postgres:мойпароль@localhost:5432/fires

## Если что-то не работает
| Проблема | Решение |
|----------|---------|
| "Python не найден" | Установите Python, перезапустите setup.bat |
| "Ошибка подключения к БД" | Проверьте что PostgreSQL запущен, проверьте .env |
| Браузер не открылся | Откройте вручную: http://127.0.0.1:8000 |
| Порт 8000 занят | В .env измените APP_PORT=8001 |

## Структура папок
- **data/uploads/** — сюда загружаются файлы данных
- **data/results/** — здесь сохраняются результаты анализа
- **.env** — настройки подключения (не передавать другим людям)
